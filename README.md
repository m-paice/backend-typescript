# Backend em typescript

## Requisitos

-   Nodejs
-   Docker

## Instalação

```bash
docker-compose up
npm start
```

## Rotas

Listar todos os autores

GET /author

```bash
http://localhost:7878/author
```

Criar um autor

POST /author

```bash
http://localhost:7878/author
```

Deletar um autor

DELETE /author/:id

```bash
http://localhost:7878/author/:id
```
